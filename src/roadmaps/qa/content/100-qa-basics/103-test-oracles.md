# Test Oracles

テストオラクルはメカニズムです。
テストケースに対するプログラムの出力の正確さをチェックするために使用できるプログラム自体とは異なります。
概念的には、テストケースがテストオラクルとテスト中のプログラムに与えられるプロセスをテストすることを考えることができます。
